## 自定义网络

> 查看所有的docker网络

![在这里插入图片描述](https://img-blog.csdnimg.cn/20200814105640929.png#pic_center)

**网络模式**

bridge： 桥接模式，桥接 docker 默认，自己创建的也是用brdge模式

none： 不配置网络

host： 和宿主机共享网络

container：容器网络连通！（用的少， 局限很大）

**测试**


4. 编写Dockerfile

```shell
FROM java:8

COPY *.jar /app.jar

CMD ["--server.port=8080"]		# 参数列表格式：CMD ["参数1", "参数2"...]。在指定了 ENTRYPOINT 指令后，用 CMD 指定具体的参数。

EXPOSE 8080

ENTRYPOINT ["java", "-jar", "/app.jar"]
```
